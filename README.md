# tictac7

Tic-Tac-Toe game expanded to 7x7 field

## Project Setup

### Prerequisites

- Node.js at least version 9.6.7 - You can download it here: [nodejs.org/en](https://nodejs.org/en)

### Installation

Run:

```bash
npm install
```

### Deployment

To start it on localhost run:

```bash
npm start
```

To generate a 'dist' directory run:

```bash
npm run build
```
